# Сайт регистрации пациентов - Медиленд

Этот сайт предназначен для регистрации пациентов в реабилитационном центре "Медиленд".

## Структура проекта

- `index.html` - Главная страница с формой регистрации
- `styles.css` - Стили для сайта
- `script.js` - JavaScript для обработки формы
- `bot-server.js` - Сервер Telegram-бота для сотрудников
- `package.json` - Зависимости для сервера бота
- `subscribers.json` - База зарегистрированных сотрудников
- `images/` - Папка для изображений (логотип)
- `CNAME` - Для домена mediland.kg

## Telegram-уведомления для сотрудников

Проект теперь поддерживает запуск на Vercel через webhook. Сайт и API будут развернуты вместе, а форма отправляет заявку на `/api/notify`.

### Настройка на Vercel

1. Залейте репозиторий на GitHub.
2. Подключите его к Vercel.
3. В настройках проекта Vercel задайте переменные окружения:
   - `TELEGRAM_BOT_TOKEN` — токен вашего бота
   - `GITHUB_TOKEN` — GitHub Personal Access Token с правом записи в репозиторий

   Если Vercel не определяет репозиторий автоматически, добавьте также:
   - `GITHUB_OWNER` — имя владельца репозитория
   - `GITHUB_REPO` — имя репозитория

4. После деплоя откройте:
   `https://<ваш-проект>.vercel.app/api/telegram?setup=1`

   Это установит webhook Telegram на ваш Vercel URL.

### Локальный запуск

Для локальной разработки работает старый `bot-server.js`:

```bash
npm install
set TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН
npm start
```

### Команды бота

- `/регистрация <пароль> <имя>` — создать аккаунт
- `/войти <имя> <пароль>` — войти и начать смену
- `/начать_смену` — включить смену
- `/закончить_смену` — завершить смену
- `/статус` — посмотреть статус
- `/отписаться` — удалить себя из списка

### Как работает

- Заявка с сайта отправляется на `/api/notify`.
- Если есть сотрудники на смене, уведомление сразу уходит им.
- Если никого нет на смене, заявка откладывается в очередь и придёт при следующем входе сотрудника.

## Функционал

- Форма регистрации с полями: ФИО, дата рождения, телефон, email, тип реабилитации, дополнительная информация
- Поддержка регистрации сотрудников: должность, отдел и табельный номер
- Простая валидация полей
- Отправка уведомлений в Telegram
- Сообщение об успешной отправке (в реальном проекте нужно подключить backend для обработки данных)

## Telegram

- В `script.js` укажите валидный `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_IDS`
- Бот должен быть добавлен в нужный чат или группу
- Если уведомления не приходят на других устройствах, проверьте настройки уведомлений в Telegram и правильность `chat_id`
